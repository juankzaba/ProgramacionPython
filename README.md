#Python - Portafolio de Aprendizaje

Este repositorio documenta mi ruta de aprendizaje desde fundamentos de Python hasta desarrollo de aplicaciones con bases de datos, interfaces gráficas, frameworks web y tecnologías frontend/backend.

## Objetivo

Construir un portafolio práctico con ejercicios, mini proyectos y proyectos finales que demuestren mi progreso como desarrollador Python y futuro profesional en datos, automatización y desarrollo web.
